# hospital-management-system
A Hospital Management System built to manage patients, doctors, appointments, billing, and medical records efficiently. Designed to simplify hospital operations with an organized and user-friendly interface.
🏥 Hospital Management System (C Programming)

This project is a console-based Hospital Management System developed using C Programming Language.

The system helps manage hospital operations such as:

Patient registration

Doctor management

Appointment booking

Billing system

Record storage using file handling

This project demonstrates the use of structures, file handling, functions, and menu-driven programming in C.

🚀 Features

👨‍⚕️ Add / View Doctor Details
🧑‍🤝‍🧑 Add / Search Patient Records
📅 Book Appointments
💳 Generate Bills
📁 Store Data in Files
📋 Menu-Driven Interface

🛠️ Concepts Used

C Programming

Structures (struct)

Functions

File Handling (fopen, fwrite, fread)

Conditional Statements

Loops

Switch Case

Arrays
